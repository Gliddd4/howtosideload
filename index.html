How to create your own DNS

Creating a NextDNS Account

First, you'll need to create an account with NextDNS:

  1. Open the NextDNS website
  2. Click on the Sign Up button
  3. Enter your credentials

Recommended: If you don't want to use your own email, you can use a temporary one instead!

  1. Go to Temp-Mail
  2. Copy your temporary email after it loads
  3. Use this email when signing up for NextDNS

2. Set Up Your Deny List

After creating your account, you'll need to add these urls to the deny list tab. This will block Apples servers from revoking you.:

  • ocsp.apple.com
  • ocsp2.apple.com
  • valid.apple.com
  • crl.apple.com
  • certs.apple.com
  • appattest.apple.com
  • vpp.itunes.apple.com

3. Set Up Your Allow List

You'll need to add these urls to the allow list tab. This will allow you to use Feather and ChatGPT.

  • register.appattest.apple.com
  • app.localhost.direct
